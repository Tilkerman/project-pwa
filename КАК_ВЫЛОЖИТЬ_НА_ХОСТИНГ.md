# Как выложить приложение на хостинг

## Шаг 1: Собрать приложение

1. Откройте командную строку (cmd)
2. Перейдите в папку проекта:
   ```
   cd C:\Users\z108\PWA\priv
   ```
3. Соберите приложение:
   ```
   npm run build
   ```
4. Дождитесь завершения - должно появиться сообщение "build complete"

После сборки файлы будут в папке `dist/`

---

## Вариант 1: Netlify (САМЫЙ ПРОСТОЙ)

### Что нужно:
- Email адрес (любой)

### Как сделать:

1. **Откройте сайт:** https://www.netlify.com/
2. **Нажмите "Sign up"** (Регистрация)
3. **Войдите через GitHub** (или создайте аккаунт)
4. **На главной странице найдите кнопку "Add new site" → "Deploy manually"**
5. **Перетащите папку `dist`** из вашего проекта в окно браузера
6. **Готово!** Netlify даст вам адрес типа `ваше-приложение.netlify.app`

### Ваш сайт будет доступен по адресу:
`https://ваше-приложение.netlify.app`

---

## Вариант 2: Vercel (тоже простой)

1. **Откройте:** https://vercel.com/
2. **Войдите через GitHub**
3. **Нажмите "Add New Project"**
4. **Выберите папку `dist`** из вашего проекта
5. **Нажмите "Deploy"**

---

## Вариант 3: GitHub Pages (если есть GitHub)

1. Создайте репозиторий на GitHub
2. Загрузите туда папку `dist`
3. В настройках репозитория включите GitHub Pages
4. Выберите папку `dist` как источник

---

## После выкладки:

✅ Приложение будет доступно по адресу типа `https://ваше-приложение.netlify.app`
✅ Работает на всех устройствах (компьютер, телефон, планшет)
✅ Можно установить как PWA приложение
✅ Все данные хранятся локально на устройстве пользователя

---

## Важно:

После выкладки на хостинг, если вы измените код:
1. Снова выполните `npm run build`
2. Загрузите обновленную папку `dist` на хостинг

## ⚠️ УСТРАНЕНИЕ ПРОБЛЕМЫ "БЕЛЫЙ ЭКРАН" НА МОБИЛЬНОМ УСТРОЙСТВЕ

Если после загрузки приложения на мобильном устройстве виден только белый экран:

### Причина:
Старый Service Worker кэшировал файлы с неправильными путями или произошла ошибка загрузки.

### Решение:

#### Шаг 1: Очистка кэша и Service Worker на мобильном устройстве

**Для Android (Chrome):**
1. Откройте Chrome на телефоне
2. Перейдите на ваш сайт
3. Нажмите на меню (три точки) → **Настройки**
4. Перейдите в **Сайты** → найдите ваш сайт
5. Нажмите на ваш сайт → **Очистить и сбросить**
6. Или: **Настройки** → **Конфиденциальность и безопасность** → **Очистить данные просмотра** → выберите ваш сайт → **Очистить данные**

**Для iOS (Safari):**
1. Откройте **Настройки** на iPhone
2. Перейдите в **Safari**
3. Нажмите **Очистить историю и данные веб-сайтов**
4. Подтвердите очистку

#### Шаг 2: Удаление установленного PWA

**Android:**
1. Найдите иконку приложения на главном экране
2. Зажмите иконку → **Удалить** или перетащите в корзину

**iOS:**
1. Зажмите иконку приложения на главном экране
2. Нажмите на крестик (X) для удаления

#### Шаг 3: Перезагрузка страницы

1. Откройте сайт в браузере заново
2. Нажмите **Ctrl+F5** (на компьютере) или **обновить** (на мобильном)
3. Или закройте и откройте браузер заново

#### Шаг 4: Переустановка PWA

1. Откройте сайт в браузере
2. В меню браузера выберите **"Добавить на главный экран"** или **"Установить приложение"**
3. Подтвердите установку

### Если проблема не решена:

1. **Проверьте консоль браузера** (на компьютере через F12)
2. **Убедитесь, что все файлы загружены** на хостинг из папки `dist`
3. **Проверьте, что приложение собрано правильно** - выполните `npm run build` заново
4. **Убедитесь, что на хостинге нет старого кэша** - возможно, нужно очистить кэш на стороне хостинга
