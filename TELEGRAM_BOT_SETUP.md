# Настройка Telegram бота для разработчика

## Важно для разработчика:

Этот файл содержит инструкции для **разработчика приложения**. Пользователям не нужно создавать бота - они только получают свой Chat ID.

## Шаг 1: Создание бота (один раз)

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Введите имя бота (например: "Трекер Привычек")
   - Введите username бота (должен заканчиваться на `bot`, например: `habit_tracker_bot`)
4. BotFather отправит вам **токен бота** (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. **Скопируйте токен** - он понадобится для настройки

## Шаг 2: Настройка токена в приложении

1. Откройте файл `src/config/telegram.ts`
2. Найдите строку: `export const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE'`
3. Замените `YOUR_BOT_TOKEN_HERE` на токен вашего бота
4. Сохраните файл

## Шаг 3: Настройка Service Worker

1. Откройте файл `public/sw-custom.js`
2. Найдите строку: `const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE'`
3. Замените `YOUR_BOT_TOKEN_HERE` на тот же токен бота
4. Сохраните файл

## Шаг 4: Пересборка приложения

После изменения токена необходимо пересобрать приложение:

```bash
npm run build
```

## Что делают пользователи:

Пользователям нужно только:
1. Получить свой Chat ID через @userinfobot в Telegram
2. Ввести Chat ID в настройках приложения
3. Готово! Уведомления будут приходить через ваш общий бот

## Безопасность:

⚠️ **ВАЖНО:** Токен бота будет виден в коде приложения. Это нормально для PWA без сервера, но:
- Не используйте токен от важного бота
- Создайте отдельного бота специально для этого приложения
- Если токен скомпрометирован, создайте нового бота через @BotFather

## Проверка работы:

1. Запустите приложение
2. Откройте настройки Telegram (кнопка ⚙️)
3. Введите свой Chat ID
4. Нажмите "Отправить тест"
5. Должно прийти тестовое сообщение в Telegram
