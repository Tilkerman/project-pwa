# Инструкция по проверке изменений

## Шаг 1: Перезапуск dev server

### Вариант А: Через терминал Cursor
1. Найдите внизу экрана терминал (окно с черным фоном)
2. Нажмите `Ctrl+C` (чтобы остановить текущий процесс)
3. Введите команду: `npm run dev`
4. Нажмите Enter
5. Дождитесь сообщения "Local: http://localhost:5173"

### Вариант Б: Если терминал закрыт
1. В Cursor нажмите `Ctrl+` (обратная кавычка) или `View → Terminal`
2. Введите: `cd C:\Users\z108\PWA\priv`
3. Нажмите Enter
4. Введите: `npm run dev`
5. Нажмите Enter

## Шаг 2: Очистка кэша браузера

### В Chrome/Edge:
1. Откройте приложение в браузере
2. Нажмите `Ctrl+Shift+Delete`
3. Выберите "Изображения и файлы в кэше"
4. Нажмите "Удалить данные"
5. Или просто нажмите `Ctrl+F5` на странице (жесткая перезагрузка)

### В Safari (iPhone):
1. Закройте приложение полностью
2. Откройте заново

## Шаг 3: Проверка консоли браузера

### На компьютере:
1. Откройте приложение в браузере
2. Нажмите `F12` (или правой кнопкой мыши → "Проверить элемент")
3. Откройте вкладку "Console" (Консоль)
4. Кликните на кнопку с "+" в палитре цветов
5. Должны появиться сообщения:
   - `openColorPicker called`
   - `showColorPickerModal: true`

### На iPhone:
1. Подключите iPhone к Mac
2. В Safari на Mac: Develop → [Ваш iPhone] → [Ваш сайт]
3. Или используйте Safari на iPhone с включенным веб-инспектором

## Шаг 4: Что должно появиться

После клика на кнопку с "+" (последняя кнопка в палитре цветов с градиентом):
- Должно открыться модальное окно с большим цветовым селектором
- Внизу должен быть ползунок с радужными цветами
- Должно быть поле для ввода hex-кода цвета

## Если ничего не работает:

1. Проверьте, что dev server запущен (должно быть сообщение "Local: http://localhost:5173")
2. Проверьте консоль на ошибки (красные сообщения)
3. Убедитесь, что вы открыли правильный URL (обычно http://localhost:5173)
